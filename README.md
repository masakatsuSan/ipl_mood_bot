## 💡 What is this?

**IPL AI Mood Bot** is a real-time cricket fan companion app built during the **GDG Siliguri Agentic Premier League** hackathon.

You type what's happening in an IPL match — and Google Gemini AI instantly responds with:

| Output | Description |
|--------|-------------|
| 🎙️ **Live Commentary** | Dramatic, exciting play-by-play like a real commentator |
| 📊 **Match Mood** | 🔥 Intense / 😎 Comfortable / 😬 Tense |
| 🔮 **5-Over Prediction** | Bold AI prediction for what comes next |

---

## 🎯 The Problem It Solves

2.5 billion cricket fans watch IPL worldwide. But:

- Casual fans don't always understand what's happening
- Commentators repeat the same lines
- There's no personalized, real-time AI fan experience

**IPL AI Mood Bot** is that missing AI friend who sits next to every fan and makes the match more exciting, accessible, and intelligent.

## ⚙️ How It Works

```
User types match situation
        ↓
Gemini 2.5 Flash API receives prompt
        ↓
AI analyzes context + emotion
        ↓
Returns: Commentary + Mood + Prediction
        ↓
Displayed instantly on screen
```

## 🏃 Getting Started

### 1. Get a Free Gemini API Key
- Go to [aistudio.google.com](https://aistudio.google.com)
- Sign in with your Google account
- Click **Get API Key** → **Create API Key**
- Copy the key

### 2. Run the App
```bash
# Just download and open — no install needed!
git clone https://github.com/masakatsuSan/ipl-ai-mood-bot.git
cd ipl-ai-mood-bot
open ipl_mood_bot.html   # or double-click the file
```

### 3. Use It
- Paste your Gemini API key in the top box → click **Save**
- Type what's happening in the match
- Hit the **→** button or press **Enter**
- Watch Gemini react! 🏏
